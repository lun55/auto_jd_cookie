# 提取Cookie的简易APP
- JD 账户保存
- cookie提取
- 发送至服务器

发送数据格式：
{
  "userName": "XXXXXXX",
  "device": "iPhone 13 Pro",
  "cookie": "pt_key=xxx;pt_pin=abc123;",
  "time": "2025-05-22 14:30:00",
}
